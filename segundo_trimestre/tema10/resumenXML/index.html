Resumen XML


Es un lenguaje de marcas cuyo propósito es compartir datos a través de diferentes sistemas.

Las características de XML son:

  • Las etiquetas se definen
  • Consta de dos partes: Prólogo y Cuerpo
  • Utiliza espacios de nombres para evitar ambiguedades
  • Misma sintaxis que en HTML
  • Para que este bien formado, debe cumplir las reglas w3c
  • Usa esquemas en forma de arbol con una unica raiz
  • Se validan los documentos

Estructura documento XML

Consta de dos partes:

  • Prologo:
    1. Declara el XML
    2. Información del documento compuesta por:
      • Espacio de nombres
      • Hojas estilo XSLT
      • Hojas estilo CSS
  • Cuerpo:
    1. Arbol del contenido
    2. Unica raiz

Sintaxis XML

  • Elemento:
    • Es fundamental
    • Tiene que tener su etiqueta de apertura y de cierre
    • Contiene un atributo con su valor
    • Contiene un contenido
  • Elementos vacios:
    • Se define: < elemento />

Formacion de documentos

Normas básicas son:

  • Prólogo definido
  • Estructura de arbol con una unica raiz
  • Etiquetas emparejadas: apertura y cierre
  • Los atributos entre comillas
  • Se distingue las mayusculas de las minusculas
  • Las estiquetas se nombran
  • Los comentarios van fuera de las etiquetas

Validacion de documentos

Hay dos formas:

  1. DTD
  2. XML Schema

DTD

Describe la estructura,es decir, los elementos y el orden. Se puede ubicar de tres maneras distintas:

  1. Interna
  2. Externa
  3. Mixta
