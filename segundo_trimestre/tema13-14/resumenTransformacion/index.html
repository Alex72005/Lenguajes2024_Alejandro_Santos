Transformaciones XSL

Este tema abarca el uso de XSLT para transformar documentos XML, incluyendo su integración con XPath y XSL-FO, así como herramientas de procesamiento.

Hojas de estilo XSLT

  • Elemento Raiz: <stylesheet>
  • Asociación de la hoja de estilo XSLT: Se asocia al documento XML en el elemento raíz

Proceso de Transformación XSLT

  • Definición de plantillas: Uso de <stylesheet> y <xsl:template> para definir transformaciones.
  • Selección de nodos: Mediante <xsl:apply-templates> y XPath.
  • Condiciones: Uso de <xsl:if> y <xsl:choose>.
  • Iteraciones: <xsl:for-each> para recorrer nodos.

Herramientas de Procesamiento

  • Oxygen XML Editor: Editor avanzado para XSLT.
  • Saxon: Procesador XSLT open-source.
  • Xalan: Otra alternativa open-source.
  • Online XSLT Test Tool: Herramienta online para pruebas rápidas.
