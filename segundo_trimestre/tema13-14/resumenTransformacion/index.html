Transformaciones XSL

Este tema abarca herramientas y lenguajes para transformar y trabajar con documentos XML, como XPath, XSLT y formatos de intercambio de datos como JSON, CSV y YAML.

Lenguajes de Transformación XML

  • XPath: Selección y navegación de nodos en documentos XML.
  • XSLT: Transformación de XML a otros formatos como HTML.
  • XSL-FO: Generación de otros formatos como PDF.

XPath

Un lenguaje para navegar por el árbol de un documento XML:

  • Propósito: Acceso a nodos, extracción de datos y realización de cálculos.
  • Sintaxis: Usa rutas jerárquicas (/ y //) y filtros avanzados ([]).
  • Nodos: Raíz, elementos, atributos, texto y más.

Formatos de Intercambio de Datos

Diferentes formatos para compartir información entre sistemas:

  • JSON: Ligero, basado en pares clave-valor, extensamente utilizado en APIs REST.
  • CSV: Formato tabular, ideal para hojas de cálculo.
  • YAML: Estructura jerárquica y legible por humanos, usado en configuraciones.

Comparativa de Formatos

  • XML: Extensible y jerárquico, con validación mediante XSD.
  • JSON: Ligero y fácil de procesar en aplicaciones web.
  • CSV: Sencillo y ampliamente compatible con herramientas de datos.
  • YAML: Menos verboso que JSON, ideal para configuraciones.
